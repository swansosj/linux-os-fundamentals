404

Page Not Found

What happened?

The server received your HTTP request, but the file you asked for doesn't exist in the document root at /var/www/lab.local/.

This is an HTTP 404 response — the server is working fine, it just can't find that specific file.

Things to check:

  • Is the URL spelled correctly?
  • Does the file exist? Check: ls -la /var/www/lab.local/
  • Are the file permissions correct? The www-data user needs read access.
  • Check the error log: sudo tail /var/log/nginx/lab.local.error.log

See it in the logs:

$ sudo tail -1 /var/log/nginx/lab.local.error.log
open() "/var/www/lab.local/your-page" failed (2: No such file or directory)

$ sudo tail -1 /var/log/nginx/lab.local.access.log
192.168.1.50 - - [01/Mar/2026:15:05:00] "GET /your-page HTTP/1.1" 404 ...
