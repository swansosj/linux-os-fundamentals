What Makes This Page Possible?

This simple web page required several Linux services working together. Here's the chain of events that brought this page to your screen:

Your Browser │ │ 1. "I need to find www.lab.local" │ ▼ BIND9 DNS Server (port 53) │ │ 2. "www.lab.local = 192.168.1.100" │ ▼ Your Browser │ │ 3. TCP connection to 192.168.1.100:80 │ 4. HTTP GET / HTTP/1.1 │ ▼ Nginx Web Server (port 80) │ │ 5. Reads /var/www/lab.local/index.html │ 6. HTTP 200 OK + HTML content │ ▼ Your Browser │ │ 7. Parses HTML, requests css/style.css │ 8. Renders the page you're seeing now │ ▼ This Page ✓

Both BIND9 and Nginx are managed by systemd — the init system (PID 1) we identified in lesson 6.1.1. Use systemctl status bind9 and systemctl status nginx to see them running.

Quick Lab Commands

# Check if this web server is running
$ sudo systemctl status nginx

# Check if DNS is running
$ sudo systemctl status bind9

# Test DNS resolution
$ dig @localhost www.lab.local +short
192.168.1.100

# View live access logs
$ sudo tail -f /var/log/nginx/lab.local.access.log

# View live error logs
$ sudo journalctl -u nginx -f
